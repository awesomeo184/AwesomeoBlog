# 1장 - 컴퓨터 내부의 언어체계

**모든 언어는 기호의 집합**이다. 하지만 의사소통을 위해서는 기호만으로는 충분치 않다. 의사소통 당사자들이 같은 **맥락(context)**을 공유하고 있어야 제대로된 의사소통이 가능하다.
